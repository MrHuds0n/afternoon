Jumbotron

Jumbotron

Jumbotron is a large heading to be used at the top of the page. You can add alt class to shift colours.

<div class="jumbotron alt" style="background-image: url('...')">
	<h1>Jumbotron</h1>
</div>

Input groups

Input groups add tidy border radii for visual grouping of certain input elements (although the input groups work on any element).

<form>
	<div class="form-control">
		<div class="input-group">
			<button class="lg">Alpha</button>
			<button class="lg">Beta</button>
			<button class="lg">Gamma</button>
		</div>
	</div>
 
	<div class="form-control">
		<div class="input-group">
			<input type="text" placeholder="Username..." />
			<button>Login</button>
		</div>
	</div>
</form>

Card grids

Card grids spread a variety of rich elements in an organised grid.

Card containers have three preset sizes: lg will not collapse, md will collapse once (to 50% width) and sm will collapse twice (to 33% and 50% width).

Within cards, p elements fill the entire empty space. Apart from that, there is no predefined behaviour and you can do with them whatever you like.

You can add the small class to individual cards to give them small text size.

Any remaining cards in the last row are automatically stretched to fill the screen.

Alpha

This is a regular card with just text.

Beta

This card also has a button.

Gamma

This card features small text.

Delta

This card will fill the bottom of the screen if your viewport is large enough.

<div class="cards sm">
	<div class="card">
		<h5>Alpha</h5>
		<p>...</p>
	</div>
	<div class="card">
		<h5>Beta</h5>
		<p>...</p>
		<button class="primary fill">Nice button!</button>
	</div>
	<div class="card small">
		<h5>Gamma</h5>
		<p>...</p>
	</div>
	<div class="card">
		<h5>Delta</h5>
		<p>...</p>
	</div>
</div>
