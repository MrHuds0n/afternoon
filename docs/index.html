This is Rhazp.

Tired of flat design everywhere, Material cropping into every project and tiled cards around every corner? Bored of using grid-based frameworks, which make you feel like building the same app every time?

Rhazp (Rhazp has zero punctuation) is a modern CSS and JavaScript UI framework built on Stylus. It features conservative, no-bullshit design, responsiveness, lightweight operation, sensible defaults and easy customisation.
